# GoSlide — Landing Page Practice

## О проекте
Учебный проект по верстке адаптивного лендинга. Работа выполнена по видеоурокам канала **"BrainsCloud"**. В ходе проекта я практиковал навыки создания современных слайдеров и адаптивной сетки.

## 🛠 Стек технологий
* **HTML5** (семантическая разметка)
* **CSS3** (Flexbox и Grid для построения макета)
* **Адаптивная верстка** (Media Queries для мобильных устройств)
